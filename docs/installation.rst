============
Installation
============


Stable release
--------------

To install full async web app example, run this command in your terminal:

.. code-block:: console

    $ pip install full_async_web_app_example

This is the preferred method to install full async web app example, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for full async web app example can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/beetleman/full_async_web_app_example

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/beetleman/full_async_web_app_example/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/beetleman/full_async_web_app_example
.. _tarball: https://github.com/beetleman/full_async_web_app_example/tarball/master
